AUIGrid Features

  • AUIGrid 는 Active-X 사용 없이 자바스크립트와 HTML, CSS 로 작성된 웹 데이터그리드(DataGrid) 입니다.
  • AUIGrid 는 jQuery 또는 기타 JS Framework 와 독립적으로 작성되었습니다. 따라서 다양한 환경에서 사용할 수 있습니다.
  • AUIGrid 는 특정 서버 사이드(PHP, JSP, ASP, .NET)에 종속되지 않습니다. 어떤 서버 사이드, WAS 든 사용 가능합니다.

단, 스파크라인(SparkLine)차트는 IE9 이상에서 사용 가능합니다.
